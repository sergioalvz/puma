= Puma
Sergio Álvarez <sergioalvarezsuarez@gmail.com>

The *Puma* project is focused on analyzing large collections of tweets in order to obtain useful information for finding location indicative words.

This project is part of the Sergio Álvarez's Master Thesis for content-based tweets' geolocation.
